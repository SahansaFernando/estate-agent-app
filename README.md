# Estate Agent React App
client-side web application that enables a user to search properties by:
•type (house, flat, any)
•price (min price, max price)
•bedrooms (min bedroom, max bedroom)
•date added (after specified date or between two given dates)
•postcode area (1st part of the postcode, e.g. BR1, NW1)

## Live Demo

npm run dev
Open http://localhost:5173

- Multi-criteria property search
- Favourites with drag-and-drop
- Responsive layout
- Jest testing
